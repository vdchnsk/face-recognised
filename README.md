# 🎭 Face_detector

**Version 0.9.0**

📃 App which is able to find all the faces on some photo, video or even webcam, and define if it's smiling

## ⚙ Technologies

### 💅 Front-end

### 🧠 Back-end

- Python
- OpenCV

## 🚀 Launching Guide

1.  Clone The repo
2.  Change the working directory on the app
3.  Launch via "python main.py"
